# Chinnam Goutham Nischay Portfolio

Static portfolio website built for GitHub Pages.

## What is included

- Resume-style portfolio sections for summary, experience, education, skills, achievements, and professional development.
- Certificate gallery linked directly to the files in this folder.
- Responsive layout with no build step required.

## Deploy to GitHub Pages

1. Create a new GitHub repository.
2. Copy the contents of this folder into the repository root.
3. Push to GitHub.
4. In the repository settings, enable GitHub Pages from the `main` branch and the root folder.
